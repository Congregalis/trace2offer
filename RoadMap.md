## Phase 1: 数据洞察与可视化 🎯 **当前重点**

**目标**：从"记录工具"进化为"智能助手"

### 1.1 核心统计仪表板
- [x] 转化率漏斗可视化 (`new → preparing → applied → interviewing → offered`)
- [x] 各状态数量统计卡片
- [x] 每周/月新增线索趋势图
- [x] 平均停留时间分析

### 1.2 Agent 数据洞察能力
- [x] 基于数据的主动建议（如："你在面试阶段停留过久，建议跟进"）
- [x] 个性化求职策略推荐
- 注：先提供能力，然后根据 phase 2 的提醒系统/心跳系统进行主动建议

## 决策记录

### 2026-03-16: 确定 Phase 1 为当前重点
- **决策**：优先实现数据洞察仪表板
- **原因**：
  1. 差异化价值最高，从"记录工具"进化为"智能助手"
  2. 技术可行，基于现有数据即可实现
  3. 为 Agent 智能化提供数据基础
  4. 可视化进展能激励用户更积极使用
